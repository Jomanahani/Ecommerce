## Installation
  - In Your terminal write :
    git clone `https://github.com/Jomanahani/Ecommerce.git`.
  - Then `cd Ecommerce`
  - Then `code .` to open theproject in vs
  - `$ npm install` This installs the required dependencies
  - To run the React project.`$ npm start`
